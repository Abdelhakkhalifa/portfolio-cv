# Portfolio / CV — Abdelhak KHALIFA

Site portfolio réalisé avec **Bootstrap 5** et **JavaScript** (projet individuel).

## Fonctions JavaScript
- Rendu dynamique des sections à partir d'un fichier `data.js` (JSON)
- Filtre de compétences (recherche en temps réel)
- Mode sombre (préférence sauvegardée en localStorage)
- Formulaire de contact avec validation + sauvegarde locale
- Export PDF via `window.print()` + styles print

## Hébergement (GitHub Pages)
1. Push du projet sur GitHub
2. Settings → Pages → Deploy from branch → `main` / root
3. Lien généré automatiquement

## Structure
- `index.html` : structure Bootstrap + sections
- `assets/js/data.js` : données du CV
- `assets/js/main.js` : logique JS
- `assets/css/style.css` : design + mode sombre + print
